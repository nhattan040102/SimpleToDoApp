# Getting started

## Set up enviroment
- Install nodejs and npm 
```
    brew install node
```

- Check node and npm version
```
    node -v
    npm -v
```

- Instal expo
```
    npm install -g expo-cli
``` 

## Installation

- Clone project from Github
    ```
    git clone https://github.com/nhattan040102/SimpleToDoApp.git
    ```
- Run app
    ```
    npm start
    ```
- Using expo app to run the app on your phone
